# Notification-Sender

Spring boot microservice that can send messages to user's phone or email. 
It uses default Java API to send email and Twillio API to send to the user's phone number.

## Technologies used:

```
$ Java
$ Spring(Core, MVC REST, Data, Eureka Cloud)
$ Postgresql
$ Docker
$ Apache Kafka
$ Java Mail API
$ Twillio API
```
